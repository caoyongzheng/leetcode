### **78. Subsets**

Given a set of **distinct** integers, nums, return all possible subsets (the power set).

For example,
Given `[1,1,1,2,2,3]` and k = 2, return `[1,2]`.


#### **Note:**
The solution set must not contain duplicate subsets.

For example,
If **nums** = `[1,2,3]`, a solution is:
```
[
  [3],
  [1],
  [2],
  [1,2,3],
  [1,3],
  [2,3],
  [1,2],
  []
]
```
